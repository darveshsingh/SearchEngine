    Big File Drop Box

    Tags filesharing

    Large files should not be sent through e-mail system, which has a limit on the combined file size of attachments and total size of the message. Instead, you can either place them on your OneDrive and share with specific people, or you can use our Big File Drop Box service which uses a combination of e-mail and FTP services to securely share files with others. For more information, please see Related Articles section on the right.

    To access this service, launch your favourite Web browser and open this URL: uwindsor.ca/bfdb

    You will be asked to sign in with your UWin Account.

    Requesting Assistance

    Have a question about this service? Ask it here.

    To request assistance with any Big File Drop Box related issues, please click on the Open Ticket button on the right. This will bring up a form that you will need to fill-out and submit to open a support ticket with the I.T. Service Desk. You will receive a confirmation e-mail with the ticket reference number and additional instructions.
