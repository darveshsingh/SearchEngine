    How can someone who is not affiliated with the University send me a large sized file over the Internet?

    Tags file sharing

    Files that are up to 10 MB in size can be sent to you using e-mail. Just keep in mind that the 10 MB restriction applies to the total size of the e-mail message, not just the attachment(s) size.

    IT Services offers a service called "Big File Drop Box" that allows users to upload a file up to 1 GB in size to our FTP server so that they can be conveniently and securely downloaded by a designated person using a Web browser. However, in order to use this service, a person is required to sign in using their UWin ID and password. Since a person who is not affiliated with the University will not have a valid UWIn ID, you will need to take an extra step to allow this person to use the Big File Drop Box.

    1. Launch your browser and open this URL: https://www.uwindsor.ca/bfdb

    2. Sign in with your UWin ID and password

    3. Click on "Invite a Friend"

    4. Provide an e-mail address of the person that you want to "invite" to upload a file for you in the space provided and click the "Invite" button.

    Once you have clicked the "Invite" button, the person whose e-mail address you specified will receive an e-mail message that will include a URL link that he/she will need to click in order to upload the file without the need to authenticate. Your invitation will expire in 7 days.
